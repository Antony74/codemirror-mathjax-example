CodeMirror KaTeX Example

This page shows CodeMiror with LaTeX syntax highlighting and a preview rendered with KaTeX.

With JavaScript switched on in your browser the rendered notation updates as you edit.

With JavaScript switched off we find a similar textarea instead of the CodeMirror element, and the notation is not rendered dynamically, but could be rendered on the server side, as shown by the hardcoded rendering provided here.

This is vanilla JavaScript. Packages such as @uiw/react-codemirror can help CodeMirror be integrated into a framework tidily.
